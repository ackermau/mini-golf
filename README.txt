Mini Golf Parkour Game - by Austin Ackerman

Controls: 
Q - rotate camera left
E - rotate camera right
WASD - locked to original directions (trying to fix) - movement
Space - Jump

Description:
1500 possible points
5 Pickups worth 100 points each (may be loud)
9 holes worth 100 points each
1 secret hole worth 150 points
hole 9 has 3 holes worth 50, 100, and 150 points

Once started you're stuck playing forever(unless you alt+f4)

All code is in code folder
